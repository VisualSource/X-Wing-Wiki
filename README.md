## Prerequisites

- node 18._._

## Run Locally

Clone the project

```bash
  git clone https://github.com/VisualSource/X-Wing-Wiki
```

Go to the project directory

```bash
  cd X-Wing-Wiki
```

Install dependencies

```bash
  pnpm install
```

Build Application

```bash
  pnpm build
```

Start the server

```bash
  pnpm preview
```
